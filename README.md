# Document One - Government Portal

Integrated Government Document Portal with modern UI/UX

## Features

- Government-style three-row header
- Language toggle (English/Hindi)
- Accessibility features
- Responsive navigation
- Official government branding

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm start
```

Opens on [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm build
```

## Header Components

The government-style header includes:
- **Utility Strip**: Language toggle, font size controls, accessibility button
- **Branding Row**: Government emblems, state logo, MyGov logo, project title
- **Navigation Bar**: Menu items with Login/Sign Up

All styling follows Indian government portal guidelines.
